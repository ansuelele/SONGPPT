# 宋韵沉静美学 · 视觉提示词生成器

## 技能说明

本技能生成适用于 GPTImage 2.0 等 AI 绘图模型的完整视觉指令提示词。
**不直接出图**，仅输出标准化中英双语提示词文本，供其他 AI agent 或用户自行调用绘图工具。

核心美学理念：**沉静显影**（Quiet Revelation）——画面不靠清晰物件抢夺注意，而让主体、信息或产品像从一块温润、被时间浸泡过的材质中慢慢浮现。

## 视觉美学体系

以下为原创视觉语言，洗稿自宋人审美内核，非任何第三方模板：

### 材质承载层

画面底层不是空白画布，而是一块**被时间浸泡过的可触摸材质**，根据主题自动选择：

| 主题类型 | 推荐肌理 |
|----------|----------|
| 香道、茶道、器物 | 古绢纹理、旧纸纤维、漆面微光 |
| 陶瓷、工艺、非遗 | 陶土颗粒、釉面酥光、窑变肌理 |
| 书画、版本、典籍 | 楮皮纸纹、墨渍渗透、虫蛀痕迹 |
| 节气、食物、本草 | 宣纸毛边、竹纤维、草木染色 |
| 建筑、园林、空间 | 旧木年轮、石面粉化、苔痕斑驳 |
| 数据、报告、商业 | 微缩文字底图、索引纸卡、档案质感 |

### 光影逻辑

中心区域受一束**克制的漫射光**托出——如同月光透过窗纸、灯下观旧画、隔纱见远山。光不是打上去的，而是从材质内部渗出来的。边缘自然收暗，形成缓慢的旋转感和空间深度。

### 色彩体系

| 角色 | 色彩描述 | 用法 |
|------|----------|------|
| 大底 | 旧纸白、绢本米黄、瓷釉灰青、旧木褐 | 大面积铺陈，保持低饱和与纤维感 |
| 主体 | 根据知识内容变调的单色柔雾团块 | 中心略深、边缘雾化消散 |
| 点睛 | 仅占极小面积 | 在标题交界、关键数字、视觉转折处承担情绪语义 |
| 文字 | 象牙白、淡灰金、旧墨褐、雾面浅色 | 如印痕而非屏幕高亮 |
| 暗部 | 边缘自然收深 | 保留细节层次，拒绝纯黑硬切 |

点睛色根据主题变调规则：

| 主题气质 | 点睛色方向 |
|----------|------------|
| 学术、典籍、考据 | 更冷、更清洁，如微光标注 |
| 文人生活、节气、饮食 | 更温热、更油润、更近纸感 |
| 工艺、非遗、器物 | 保留材质本色，如釉面反射或漆面光泽 |
| 禅意、修行、茶香 | 更淡、更透气、几近透明 |
| 商业、品牌、发布 | 更锋利、更集中，仅占针尖大小面积 |

### 文字造型

文字是画面的**主要造型力量**，而非装饰附属。

- **中文标题**：细长清瘦、带呼吸感的宋体或书卷气质，字距略松，行距留出空气。允许词语被拆开、错位、分层排列，文字块如漂浮的笺纸或落叶，不必完全对齐却要有内在秩序。
- **英文/数字/注释**：更轻的衬线体或窄体字，作为节奏停顿和旁注，面积极小但位置精确。
- **主副对照**：大中文承担情绪与诗性，小英文/数字承担说明、时间、类别、排行、指标的理性回声。
- **禁忌**：不做可爱图标、不堆叠装饰元素、不用纯黑纯白硬切。文字如自然沉积，安静含蓄而有记忆点。

### 版式秩序

中心凝聚、四周低声补充。

- **中央**：标题、核心数据、器物轮廓、图表结论或主要视觉物
- **上方**：少量品牌/系列名/章节名/短句，位置克制造
- **两侧**：括号式注释、竖排关键词
- **底部**：极细分隔线、短英文、脚注、来源、一行密集小字压住画面

阅读路径：从中心大字进入 → 图形或重点色短暂停顿 → 落到底部信息层。
密度从中心向边缘递减，边缘信息小而稳而准。
